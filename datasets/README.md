# Datasets for Lying Style NLP Research

Research hypothesis: "The distribution of text generated by a language model when instructed to lie is noticeably different from the distribution of text the model produces under normal, truthful conditions."

## Quick Start

```bash
# From project root, with venv activated:
python datasets/download_datasets.py
```

This downloads all datasets to local disk. Large dataset files are git-ignored; only samples and scripts are tracked.

## Datasets

### 1. TruthfulQA (`truthfulqa/`, `truthfulqa_mc/`)
- **Source:** HuggingFace `truthful_qa` (generation + multiple_choice configs)
- **Paper:** Lin et al., 2022 - "TruthfulQA: Measuring How Models Mimic Human Falsehoods"
- **Size:** 817 questions across 38 categories
- **Format:** Questions with correct/incorrect reference answers
- **Columns (generation):** `type`, `category`, `question`, `best_answer`, `correct_answers`, `incorrect_answers`, `source`
- **Relevance:** Questions designed to elicit false answers from LLMs. Enables comparison of model outputs when instructed to answer truthfully vs when producing known-false responses.

### 2. CounterFact (`counterfact/`)
- **Source:** HuggingFace `NeelNanda/counterfact-tracing`
- **Paper:** Meng et al., 2022 - "Locating and Editing Factual Associations in GPT"
- **Size:** 21,919 factual statements
- **Format:** Prompts with true and false completions
- **Columns:** `prompt`, `target_true`, `target_false`, `subject`, `relation`, etc.
- **Relevance:** Directly provides paired true/false completions for factual prompts. Ideal for studying distributional differences between truthful and deceptive generation.

### 3. FEVER NLI (`fever_nli/`)
- **Source:** HuggingFace `pietrolesci/nli_fever`
- **Paper:** Thorne et al., 2018 - "FEVER: a Large-scale Dataset for Fact Extraction and VERification"
- **Size:** 248,342 claim-evidence pairs (train: 208K, test: 20K, dev: 20K)
- **Format:** Premise-hypothesis pairs with SUPPORTS/REFUTES/NOT ENOUGH INFO labels
- **Columns:** `premise`, `hypothesis`, `fever_gold_label`, `label`, etc.
- **Relevance:** Verified true/false claims with evidence. Can be used to construct prompts where LLMs generate supporting or refuting text for known facts.

### 4. LIAR (`liar/`)
- **Source:** HuggingFace `UKPLab/liar`
- **Paper:** Wang, 2017 - "Liar, Liar Pants on Fire"
- **Size:** 12,836 statements (train: 10,269, val: 1,284, test: 1,283)
- **Format:** PolitiFact statements with 6-level truthfulness labels
- **Columns:** `text`, `label_text`, `labels`, `context`
- **Labels:** pants-fire (0), false (1), barely-true (2), half-true (3), mostly-true (4), true (5)
- **Relevance:** Real-world graded truthfulness labels. Useful as prompts for LLM truth/lie generation and as a deception detection benchmark.

### 5. Azaria & Mitchell True-False Dataset (`azaria_true_false/`)
- **Source:** https://azariaa.com/Content/Datasets/true-false-dataset.zip
- **Paper:** Azaria & Mitchell, 2023 - "The Internal State of an LLM Knows When It's Lying"
- **Format:** CSV files with true/false statements across 6 topics (animals, capitals, cities, companies, elements, inventions)
- **Manual download:** If the automated script fails, download the zip from the URL above and extract to `datasets/azaria_true_false/`
- **Relevance:** Core dataset from the SAPLMA paper. Statements used to probe whether LLM internal states differ between true and false claims.

## Samples

The `samples/` directory contains 10-item JSON samples from each dataset for quick inspection without loading the full datasets.

## Loading Datasets

```python
from datasets import load_from_disk

# Load any dataset
ds = load_from_disk("datasets/truthfulqa")
print(ds["validation"][0])

ds = load_from_disk("datasets/counterfact")
print(ds["train"][0])

ds = load_from_disk("datasets/fever_nli")
print(ds["train"][0])

ds = load_from_disk("datasets/liar")
print(ds["train"][0])
```
